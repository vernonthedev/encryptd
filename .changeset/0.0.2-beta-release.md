### Features

- Core encryption engine: AES-256-GCM with PBKDF2-HMAC-SHA256 (100,000 iterations)
- Per-encryption random salt stored in payload for cryptographic uniqueness
- Type-safe TypeScript bindings via NAPI-RS with EnvPayload interface
- Cross-platform native binaries for Linux x64, macOS x64/arm64, Windows x64
- CLI commands for encrypt and decrypt operations on environment files
- Automated release pipeline with GitHub Packages publishing
