### Features

- Core encryption engine: AES-256-GCM with PBKDF2-HMAC-SHA256 key derivation at 100000 iterations
- Per-encryption random salt stored in payload for cryptographic uniqueness
- Type-safe TypeScript bindings via NAPI-RS with EnvPayload interface
- Cross-platform native binaries for Linux x64, macOS x64 and arm64, Windows x64
- CLI commands for encrypt and decrypt operations on environment files
- Automated release pipeline publishing to GitHub Packages

### Use cases

- Encrypt environment files for secure deployment across CI/CD pipelines
- Protect sensitive configuration values in shared repositories
- Decrypt at runtime in Node.js applications without exposing secrets

### Direction

- Stable encryption primitives suitable for production workloads
- Zero-dependency runtime with minimal binary size
- First-class TypeScript support with full type inference
- Distribution through GitHub Packages with beta and latest channels
